## Deskripsi Umum
Program penyelesaian puzzle Rush Hour menggunakan algoritma UCS, GBFS, A*, dan Beam Search.
## Requirement
* Java
## Cara Compile
1. Clone repository
2. Masuk ke src
```sh
cd src
```
3. Run command berikut
```sh
javac *.java
```
## Cara Run
1. Masuk ke direktori file Main.class
2. Run command berikut
```sh
java Main
```
